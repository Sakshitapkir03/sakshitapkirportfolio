PostGo

JavaJSPODBCPostgreSQLJavaScript

PostGo is a courier management system that handles the full journey of a shipment — from the moment it's picked up to when it reaches the customer. Couriers and admins can log in, look up packages, update delivery statuses, and manage routes all from a single web interface. I focused heavily on performance: optimized stored procedures in PostgreSQL made lookups significantly faster, and graph-based routing algorithms find the most efficient delivery paths between locations — which pushed overall system throughput up by 35%.
