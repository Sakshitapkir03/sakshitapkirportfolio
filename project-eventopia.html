Event Management Platform

Eventopia

TypeScriptNext.jsReactNode.jsMongoDB

Eventopia is a platform where people can create, discover, and register for events — similar to Eventbrite. Organizers can post events with all the details, manage their attendee lists, and accept ticket payments through Stripe. Attendees can browse events, register, and check out in just a few clicks. I used Clerk to handle authentication so sign-up and login are smooth from day one, and built webhook validation into the payment flow so every transaction is properly confirmed before an order is finalized.
